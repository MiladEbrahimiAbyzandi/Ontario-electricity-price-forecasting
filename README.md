# HOEP Forecasting Project

This project focuses on forecasting the Hourly Ontario Electricity Price (HOEP) using statistical, machine learning, and deep learning methods. It also incorporates Hadoop and PySpark to simulate a scalable big-data preprocessing workflow.

## Objectives
- Forecast HOEP for 1-hour, 2-hour, and 3-hour ahead horizons
- Compare statistical, ML, and DL models
- Use PySpark for preprocessing and feature engineering
- Evaluate models using MAE, RMSE, and MAPE

## Project Structure
- data/: raw, interim, and processed data
- notebooks/: experiments and analysis
- src/: reusable code
- reports/: report and outputs
- figures/: plots and visuals
